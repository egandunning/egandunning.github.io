User Stories

Functions users will expect from our program.

  1. As Haley, I want to launch the application from a shortcut.
  2. As Haley, I want to be able to see what happens to my passwords when hashed.
  3. As Joe, I want to see how long it took the break the encryption to gain insight on encryption strength.
  4. As Haley, I want to share my encryption times on Facebook.
  5. As Haley, I want to encrypt a file so only I can look at it.
  6. As Joe, I want to input a string of encrypted characters and have it decrypted to view the original string.
  7. As Joe, I want to put a time limit on the decryption process so that the program does not get stuck trying to complete a large process.
  8. As Joe, I want to take advantage of the GPU in the decryption algorithm.
  9. As Joe, I want to view a graph.
  10. As Joe, I want to be able to input my own code to change how decryption in the program works to further understand decryption.
